The Code

                    
function getValues() {

    let message = document.getElementById('userInput').value;
    
    if (message.length == 0) {
        Swal.fire({
            icon: 'error',
            backdrop: false,
            title: 'Oops!',
            text: 'Please enter a message to see if its a palindrome'
        });
    } else {

        let revMsg = checkForPalindrome(message);
        
        displayResults(revMsg);

    }

}

function checkForPalindrome(str) { 
    
    var a = str.replace(/\s|[0-9_]|\W|[#$%^&*()]/g, "").toLowerCase();

    if (a === a.split("").reverse().join("")) {
        return true;
    }
        return false;
}



function displayResults(message) {

    document.getElementById('msg').textContent = `Your message reflected is: ${message}`;

    document.getElementById('alert').classList.remove('invisible');
}

The code is structured in three functions.

The getValues function receives the words entered by the user. If the user gives an invalid input, an alert will pop up on the screen.

The checkForPalindrome function takes the received words and checks to see if its a palindrome. It will also ignore Uppercase letters, and spaces between words. This is accomplished using RegEx.

Finally, the displayResults function displays a box, informing the user if their input is a palindrome or not.
